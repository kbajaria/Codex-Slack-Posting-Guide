# Request Template

Use a prompt like this in future projects:

```text
Please post this document to my Slack channel #channel-name with a short feature announcement.

Use the authenticated Slack session on my Mac if available.
Stage the message and attachment for me, but let me do the final send.

Here is the file:
/absolute/path/to/file.pdf

Here is the audience/context:
- what changed
- why it matters
- any tone guidance
```

## Variants

### Short announcement

```text
Please post this to Slack with a short new-feature style announcement and let me press send.
```

### Leadership update

```text
Please post this to Slack with a concise leadership-ready summary and let me review before send.
```

### Release note

```text
Please post this to Slack as a release note update with the attachment staged but unsent.
```
